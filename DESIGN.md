# 猫武士数学森林 — 完整设计与课程蓝图

> 面向香港小学 P4–P6 的**猫武士主题**数学游戏。
> 以《猫武士 Warriors》故事为背景，玩家扮演一只从幼崽成长为族长的森林猫，
> 在四大领地中通过"狩猎"（解题）获取猎物，系统化学习数学。
> 兼顾**理解与趣味**，让孩子循序渐进地完成每个阶段。

---

## 一、学情分析与教学目标

### 学生画像
- 内地 P2 → 香港 P3 → 现升 P4，存在课程衔接差异。
- 核心薄弱：**混合运算 · 小数计算 · 几何图形与立体空间感**。
- 喜欢《猫武士》系列，对角色成长、领地冒险有强烈代入感。
- 学习自信偏弱，需要**即时正反馈**和**低门槛入口**重建信心。

### 教学原则（参考 Khan Academy / DragonBox / Prodigy）
1. **CPA 教学法**：具象（Concrete）→ 图像（Pictorial）→ 抽象（Abstract），每引入新概念先用图形/实物图示。
2. **掌握式学习（Mastery Learning）**：一关未达 2 星不解锁后续，确保根基牢固。
3. **脚手架递进（Scaffolding）**：每关内题目从易到难，错误后自动降一档重练。
4. **间隔重复（Spaced Repetition）**：错题自动进入"复习狩猎"，1/3/7 天后重现。
5. **即时反馈 + 逐步解析**：每题答完立刻判分，答错展示完整解题步骤。
6. **双语支持**：默认英文（香港学校主要教学语言），可一键切换繁體中文（家庭辅导用）。

### 三个年级 · 三个"季节"

| 季节 | 年级 | 副标题 | 故事主线 | 数学核心 |
|------|------|--------|----------|----------|
| **Season 1** | P3–P4 | 森林初探 *Forest Beginnings* | 幼崽 → 武士，守护雷族领地 | 四则混合、小数、平面几何、立体空间 |
| **Season 2** | P5 | 森林深处 *Deep Forest* | 武士 → 副族长，探索邻近领地 | 分数、百分数、比与比例、进阶面积/体积 |
| **Season 3** | P6 | 星族之路 *Path to StarClan* | 副族长 → 族长，追寻星族指引 | 负数、代数方程、圆、统计与概率 |

> 每个 Season 解锁新领地、新导师、新故事章节，形成"追剧式"学习动力。

## 三、游戏系统设计

### 3.1 头衔成长系统（参考 Prodigy 等级体系）

玩家积分累积解锁头衔，头衔与《猫武士》角色成长路径一一对应：

| 积分 | 头衔 | Emoji | 解锁内容 |
|------|------|-------|----------|
| 0 | 幼崽 Kit | 🐱 | Season 1 全部领地 |
| 100 | 学徒 Apprentice | 🐾 | 可使用"提示"功能 |
| 300 | 武士 Warrior | ⚔️ | Season 2 领地解锁 |
| 700 | 资深武士 Senior Warrior | 🌟 | 每日任务解锁 |
| 1,500 | 副族长 Deputy | 🛡️ | Season 3 领地解锁 |
| 3,000 | 巫医 Medicine Cat | 🌿 | 错题本/间隔重复解锁 |
| 6,000 | 族长 Clan Leader | 👑 | 全部内容 + 隐藏关卡 |

> 每次升级有庆祝动画 + 音效，营造"追番"式成就感。

### 3.2 猎物奖励系统（即时正反馈）

答对题目按难度获得猎物（fresh-kill），猎物可视化堆叠在"猎物堆"中：

| 难度 | 猎物 | 积分 | 象征 |
|------|------|------|------|
| ★ | 老鼠 Mouse 🐭 | 10 | 基础题 |
| ★★ | 田鼠 Vole 🐀 / 松鼠 Squirrel 🐿️ / 画眉 Thrush 🐦 | 15–20 | 中等题 |
| ★★★ | 兔子 Rabbit 🐰 / 鱼 Fish 🐟 | 25–30 | 挑战题 |
| ★★★★ | 考核 Boss | 30+ | 综合考核 |

- 猎物堆满一定数量可"喂养全族"（兑换装扮/特效）。
- 连续答对触发"连击 streak"加成：连对 3 题 ×1.5 倍积分。

### 3.3 每日任务与连续打卡（参考 Duolingo / Todo Math）

| 机制 | 说明 |
|------|------|
| **每日狩猎令** | 每天 3 个小任务（如"完成 1 关""答对 10 题""在溪流渔场练习"），完成各 +20 积分 |
| **连续打卡 streak** | 连续天数打卡，火焰图标 🔥 逐日增大；断签归零（可补签一次） |
| **每日免费猎物** | 每天首次登录领取 1 只免费猎物（签到奖励） |

> 这是培养**每日学习习惯**的核心机制，用"不忍心断签"的心理驱动持续参与。

### 3.4 故事章节系统（参考 Prodigy quest / 洪恩数学故事线）

每个 Season 分为 3–4 个章节（Chapter），每章对应一组关卡，通关后推进剧情：

```
Season 1 森林初探
├── Chapter 1: 幼崽的觉醒 — 完成狩猎训练场 m1–m3
│   "火心收你为徒，你要学会用四则运算磨利爪子……"
├── Chapter 2: 溪流的试炼 — 完成溪流渔场 d1–d4
│   "灰条带你去溪边捕鱼，小数的精确如同水面的涟漪……"
├── Chapter 3: 古老的智慧 — 完成古树营地 g1–g4
│   "砂风告诉你，营地里的每块石头都藏着图形的秘密……"
└── Chapter 4: 高处的视野 — 完成高地石林 s1–s5 + 全部 boss
    "蓝星带你登上高石，用立体的眼光俯瞰整个森林……"
```

- 通关一章播放一段文字剧情（导师对话），建立情感连接。
- 章节完成解锁下一章，形成"想看下一集"的动力。

### 3.5 技能树导航（参考 Khan Academy 学习路径）

领地内关卡以**技能树**形式呈现，显示前置依赖关系：

```
m1 四则复习 → m2 运算顺序 → m3 括号 → m4 两步混合 → m5 应用题
                                                    ↓
                                                  mboss 考核
```

- 未达 2 星的关卡用"锁链"标记，提示"还需练习"。
- 已掌握（3 星）的关卡显示金色爪印 🐾。
- 提供"随机狩猎"按钮，可跨关随机抽题复习。

### 3.6 错题本与间隔重复（参考 Anki 算法）

| 功能 | 说明 |
|------|------|
| **错题收集** | 答错的题自动存入错题本，记录错误次数和最近错误时间 |
| **复习狩猎** | 首页入口"复习错题"，基于间隔重复算法（1天/3天/7天）重现 |
| **弱点分析** | 统计每个知识点的正确率，低分项标红，推荐重点练习 |

### 3.7 多样化题型（参考 DragonBox 发现式学习）

不只限于数字键入，增加互动题型：

| 题型 | 说明 | 适用 |
|------|------|------|
| 数字键入 | 键盘输入答案 | 计算题 |
| 选择题 MC | 4 选 1，含图形/文字 | 概念辨析 |
| 拖拽匹配 | 拖拽图形/数字配对 | 图形识别、等值分数 |
| 填空选择 | 从选项中拖入空格 | 比较大小 > < = |
| 数线操作 | 在数线上拖动标记 | 小数/分数/负数定位 |
| 图形构造 | 拖动拼出展开图/组合图形 | 立体空间、面积 |

### 3.8 自适应难度（参考 IXL / Khan Academy）

每关内题目难度动态调整：
- 连续答对 2 题 → 下一题难度 +1 档
- 连续答错 1 题 → 下一题难度 -1 档
- 难度范围限制在该关卡设定的 ±1 档内

> 保证孩子始终处于"最近发展区"，既不无聊也不挫败。

### 3.9 家长仪表盘（参考 Khan Academy Parent Report）

- 每周学习时长、答题正确率趋势图。
- 知识点掌握雷达图（强项/弱项一目了然）。
- 错题率最高的知识点 Top 5，附推荐练习关卡。
- 可设置每日学习时间上限（保护视力）。

## 四、教学设计亮点

### 4.1 CPA 渐进可视化

每个新概念引入时先用图形/动画，再过渡到符号：

```
分数加法（异分母）:
  Step 1 [图形] 两个不同大小的饼图，直观看到"需要统一大小才能相加"
  Step 2 [图像] 通分过程用等值分数图形演示
  Step 3 [抽象] 列出算式 1/2 + 1/3 = 3/6 + 2/6 = 5/6
```

### 4.2 解题步骤逐步展示

每道题答错后展示**分步解析**，不只给答案：

```
❌ 獵物跑掉了！
正確答案: 23

解析:
  Step 1: 先算乘法  6 × 3 = 18
  Step 2: 再算加法  5 + 18 = 23

💡 提示: 先乘除後加減
```

### 4.3 猫武士情境化应用题

所有应用题融入猫武士世界观，增强代入感：

- "一次巡逻猎到 7 只老鼠和 4 只松鼠，每只松鼠喂饱 2 只猫……"
- "巫医采了 5 把草药，每把 8 片叶子，用掉 15 片……"
- "边界长 60 个狐长，4 个巡逻队各走 8 个狐长……"

> 用"狐长""叶把""巡逻队"等猫武士术语，让数学题变成森林冒险。

### 4.4 即时鼓励语

答对后随机出现猫武士风格的鼓励：
- "好身手！" / "動作真快！" / "像個真正的武士！" / "雷族以你為榮！"

## 五、技术架构

### 技术选型
- **PWA（渐进式 Web 应用）**：HTML5 + CSS3 + 原生 JavaScript，双击 `index.html` 即可运行，无需构建/服务器；托管后可安装、离线可用。
- **持久化**：`localStorage`（多用户档案隔离）。
- **渲染**：SVG / CSS 3D / Canvas（几何模块图形）；emoji + CSS 动画，零外部图片资源。
- **响应式**：桌面 / 平板 / 手机自适应，触摸友好。
- **双语**：默认 English，可切换 繁體中文。

### 目录结构
```
瑶瑶趣味数学/
├── index.html              # 入口
├── DESIGN.md               # 本设计文档
├── manifest.webmanifest     # PWA 清单
├── sw.js                    # Service Worker（离线缓存）
├── css/style.css           # 全部样式
├── js/
│   ├── i18n.js             # 双语术语 + 头衔/猎物/领地/关卡/导师/徽章数据
│   ├── visual.js           # SVG 图形生成器（2D/3D/角度/线/展开图/数线）
│   ├── engine.js           # 题目生成器 + 判分 + 状态管理 + 存储 + 音效
│   ├── ui.js               # 界面渲染 + 交互逻辑
│   └── app.js              # 启动 + 路由
└── assets/
    └── data/               # 关卡配置（后续）
```

### 数据驱动设计

所有课程内容通过数据配置驱动，新增年级只需：
1. 在 `i18n.js` 的 `LEVELS` 数组添加关卡定义
2. 在 `engine.js` 添加对应的题目生成器函数
3. 在 `visual.js` 添加需要的 SVG 图形
